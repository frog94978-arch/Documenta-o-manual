# Primeiros Passos

Este guia rápido irá ajudá-lo a navegar e utilizar as funcionalidades básicas do sistema.

## 1. Acessando o Sistema

Para começar, acesse o sistema utilizando o endereço fornecido e faça login com suas credenciais.

- **URL:** `https://seusistema.com.br`
- **Usuário:** seuemail@provedor.com
- **Senha:** (a senha fornecida no seu e-mail de boas-vindas)

## 2. Navegando pela Interface

Após o login, você verá a tela principal. A navegação é feita principalmente pelo **menu lateral esquerdo**.

![Estrutura da tela](https://i.imgur.com/example.png)  
*Legenda: Imagem de exemplo da interface.*

- **Menu Lateral:** Contém links para todos os módulos e seções aos quais você tem acesso.
- **Barra Superior:** Mostra o nome do usuário, notificações e o botão de sair.
- **Área de Conteúdo:** Exibe a página ou funcionalidade selecionada.

## 3. Realizando sua Primeira Ação

Vamos tentar uma tarefa simples: consultar seu próprio perfil.

1.  No menu lateral, clique em **"Meu Perfil"**.
2.  A página exibirá suas informações de cadastro.
3.  Verifique se seus dados estão corretos. Se não estiverem, procure o botão **"Editar"** e corrija o que for necessário.

> [!NOTE]
> Se você não encontrar uma opção ou tiver dificuldades, entre em contato com o suporte técnico ou consulte a documentação específica do módulo.

Com isso, você já sabe o básico para navegar no sistema. Explore as outras seções para aprender mais!
